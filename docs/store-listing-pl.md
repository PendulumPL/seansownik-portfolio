# Seansownik — szkic karty Google Play

## Nazwa

Seansownik

## Krótki opis

Prywatna lista filmów i seriali z ocenami, notatkami i platformami.

## Pełny opis

Seansownik pomaga uporządkować filmy i seriale, które już oglądasz albo chcesz zobaczyć później.

Wyszukaj tytuł, przypisz go do platformy i wybierz status: obejrzane, w trakcie lub do obejrzenia. Dodaj własną ocenę i krótką notatkę, a później filtruj listę według serwisu streamingowego.

Najważniejsze możliwości:

- wyszukiwanie filmów i seriali,
- trzy czytelne listy oglądania,
- filtrowanie według platformy,
- prywatne oceny i notatki,
- numerowanie obejrzanych tytułów,
- ręczna kopia zapasowa i jej przywracanie,
- lokalne przechowywanie listy bez zakładania konta.

Seansownik nie zawiera reklam i nie wymaga rejestracji.

This product uses the TMDB API but is not endorsed or certified by TMDB.
